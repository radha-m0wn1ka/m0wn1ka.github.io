what is bash_profile file role
The .bash_profile file is a configuration file used in the Bash shell environment on Unix-like operating systems, 
  including Linux and macOS. Its primary role is to set up and customize the environment for a user when they log in 
  to the system via the command-line interface (CLI).

When a user logs in, the Bash shell reads the .bash_profile file (if it exists) to execute any commands or configurations 
  specified within it. This file allows users to customize their shell environment by setting environment variables, defining 
  aliases, modifying the PATH, and executing other commands that need to run at login.
